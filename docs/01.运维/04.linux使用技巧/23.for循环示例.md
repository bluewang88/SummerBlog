## 批量写数据
```shell
[root@OrncvW1001428 test]# for file in summer* ;do echo "123">> $file; done
[root@OrncvW1001428 test]# cat summer1
123
[root@OrncvW1001428 test]# cat summer10
123
[root@OrncvW1001428 test]# cat summer9
123
[root@OrncvW1001428 test]# 
```

## 批量更改数据
