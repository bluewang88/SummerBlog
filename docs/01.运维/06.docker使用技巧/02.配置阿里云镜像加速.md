## 1.编辑daemon.json
- 在阿里云获取加速地址
```shell
[root@node3 ~]# vim /etc/docker/daemon.json
[root@node3 ~]# sudo systemctl daemon-reload
[root@node3 ~]# sudo systemctl restart docker
```
